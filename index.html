The Light at the end of the Tunnel...

The main web application we developed for class was a Task Tracker App.

Ultilizing the skills we learned with HTML, CSS, and JavaScript we created a Node.js App that would allow the user to Insert Tasks into a database.

The main features of the application included the ability to:

  • View a list of Tasks saved in a connected Mongo Database
  • Add Tasks to the Database
  • Delete Tasks from the Database
  • Update a selected Task

Future development goals for the Task Tracker App:

  • Finish Styling the Forms and Buttons controling the Add, Delete, and Update functions
  • Add a "Find" feature to the App
  • Add animations and sound effects to the buttons
